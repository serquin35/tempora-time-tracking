# Gestión de Proyectos, Tareas y Clientes

## Proyectos
Los proyectos son el eje de organización. 
- **Creación:** Deben tener un nombre único y un código de color para identificación rápida.
- **Clientes:** Cada proyecto puede estar asociado a un cliente para facilitar la facturación.
- **Estados:** Los proyectos pueden estar activos o archivados (próximamente).

## Tareas
Dentro de cada proyecto, puedes definir hitos o tareas específicas.
- **Favoritos:** Puedes marcar tareas como favoritas para acceder a ellas más rápido desde el tracker.

## Dashboard de Proyectos
En la sección "Proyectos" puedes ver el gasto total de tiempo por cada uno y la lista de miembros asignados.
