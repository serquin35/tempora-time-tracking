# Reportes, Análisis y Exportación

## Dashboard de Reportes
- **Actividad Semanal:** Gráfico de barras que muestra el tiempo trabajado cada día de la semana actual.
- **Distribución por Proyecto:** Gráfico circular que indica qué porcentaje del tiempo se dedica a cada cliente o proyecto.

## Filtros
Puedes filtrar los registros por:
- Rango de fechas (Hoy, Semana, Mes, Personalizado).
- Proyecto o Cliente específico.

## Exportación
- **PDF:** Genera un informe visual listo para enviar a clientes.
- **Excel/CSV (Próximamente):** Para análisis externo de datos.

## Facturación
Tempora permite calcular el coste de las sesiones basándose en la tarifa por hora (si se ha configurado en el proyecto).
