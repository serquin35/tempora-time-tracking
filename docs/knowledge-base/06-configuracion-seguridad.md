# Seguridad y Configuración del Perfil

## Perfil de Usuario
- **Avatar:** Puedes subir tu propia imagen o usar el avatar generado por defecto.
- **Nombre:** Actualiza tu nombre público para los reportes de equipo.

## Seguridad
- **Cambio de Contraseña:** Recomendamos usar contraseñas de al menos 8 caracteres con símbolos. Puedes usar el icono del "ojo" para verificar que la has escrito correctamente.
- **Correo Electrónico:** Por seguridad, el correo electrónico no puede cambiarse una vez creada la cuenta.

## Preferencias
- **Temas:** Selecciona entre tema **Claro (Light)** o **Oscuro (Dark)**. Tempora se sincroniza con la preferencia de tu sistema operativo por defecto.
- **Zona Horaria:** Se detecta automáticamente para que todos los registros de tiempo sean coherentes con tu ubicación.
